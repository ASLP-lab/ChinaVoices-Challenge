# ChinaVoices Challenge 2026

ChinaVoices Challenge 2026 是面向中文多方言语音识别与中文方言种类识别的挑战赛官网。页面用于展示赛事简介、任务设置、数据说明、评测规则、重要日期、排行榜、组织者信息、报名入口和公开数据资源。

## 项目简介

本挑战赛以 NCMMSC 2026 为平台，围绕中文多方言语音识别与中文方言种类识别任务，构建统一、公开、可复现的评测流程。赛事关注低资源中文方言建模、跨方言泛化能力和真实语音场景下的系统鲁棒性。

页面包含两个主要任务：

- 中文多方言种类识别：识别语音所属的中文方言类别。
- 中文多方言语音识别：对多种中文方言语音进行文本转写。

页面包含两个排行榜：

- “限定数据方案”排行榜：用于最终正式排名与奖项评定。
- “开放数据方案”排行榜：用于展示不同数据条件下的系统性能上限。
